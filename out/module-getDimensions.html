Module: getDimensions

Get the position and dimensions of the boxes around an element's margins, borders, scrollbar gutters, padding, and content.
Parameters:
Name Type Attributes Description
element HTMLElement The element for which to calculate positions and dimensions.
relativeTo HTMLElement | Array.<number> <optional>
 Calculate position relative to an [x, y] coordinate or another element (top-left of its border box).
